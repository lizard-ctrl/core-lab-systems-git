Git Introduction

Chapter 02

Repository

The code base. This is where all project files and their revision history (like an undo history) are stored. You usually have a local repository on your computer and remote repository hosted on the internet that you are trying to keep in sync.

Branch

A version of a repository. You can have multiple branches of the same code base. A branch named `master` is created by default for a new repository.

Commit

A snapshot of a particular time in a branch's history.

git branch

List the branches in the current repository.
