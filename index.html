Git Introduction

Chapter 07

Before starting a project

If you're starting from scratch - git init

If you're copying an existing repository - git clone url/to/existing/repository

Then, connect your local repository to a remote one on Github:

  1. Create a new repository on Github if you don't have one for this project.
  2. Type git remote add origin https://path/to/your/repository.git
  3. Type git branch -M master
  4. Type git push -u origin master
