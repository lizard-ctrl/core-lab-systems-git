If you want to go back to a previous commit

  1. List the recent changes that were made to the repository - git log
  2. Identify the commit you want to go back to and copy its hash
  3. Load changes with either git checkout hash if you just want to look or with git reset hash if you want to permanently go back.

git reset

Sets the code to a particular commit and erases the current changes to the working directory. Use with caution!
