Chapter 03

git status

Shows you what is the status of your working directory: What branch are you on, what files have changes in them, what files are staged to be committed. Good way to figure out what's going on at any given moment. The output can be confusing though and takes time to get used to, but it's worth it- don't despair!

Working Directory

The most up to date version of your code on your computer. Like a document before you save it.
